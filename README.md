# 🍰 Sweet Delights Bakery Website

## 📌 Project Overview
This project is a simple business website created as part of the **Future Interns Task 3 - Full Stack Development Internship**.

The website represents a local bakery named **Sweet Delights Bakery**, showcasing its services, products, and contact information.

---

## 🏢 Business Description
Sweet Delights Bakery is a local bakery that specializes in:
- Fresh cakes
- Pastries
- Cookies
- Artisan breads

The bakery focuses on quality, freshness, and customer satisfaction.

---

## 🎯 Problem Statement
Many small local bakeries do not have an online presence, making it difficult for customers to discover their products and contact them easily.

---

## 💡 Solution
This website solves that problem by providing:
- A simple online presence
- Clear information about services
- Easy contact details
- Attractive and user-friendly design

---

## ✨ Features
- Beautiful homepage with bakery branding
- About section
- Services section
- Customer-friendly layout
- Contact information section
- Responsive and colorful design

---

## 🛠️ Technologies Used
- HTML
- CSS
